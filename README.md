# How 2 play
1. download the datapack and the resource pack
2. enable the resource pack
3. add the datapack when creating a new world

if there are issues with rendering custom mobs you shall disable all~~~~ optimisation mods like optifine 

or just play on vanilla

make sure NOT to play with shaders
